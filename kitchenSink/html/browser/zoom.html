Pinch To Zoom

The Webkit engine used by BlackBerry Smartphones or Tablets, when combined with touch-screen interfaces, supports a pinch-to-zoom gesture. This gesture can be used to rapidly change the active zoom scale of Web content rendered on the current page. However, developers may wish to block this gesture in order to control the user experience within their applications.

Pinch to zoom can be disabled by using the "user-scalable=no" property set in the viewport META tag.

<meta name="viewport" content="initial-scale=1.0,user-scalable=no">

Pinch to zoom can also be disabled by combining the "minimum-scale=1.0" and "maximum-scale=1.0" properties set in the viewport META tag.

<meta name="viewport" content="initial-scale=1.0,minimum-scale=1.0,maximum-scale=1.0">
